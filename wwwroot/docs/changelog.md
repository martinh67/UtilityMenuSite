## [1.0.0] — 2026-02-22

### Added
- **Core module: GetLastRow** — find and copy the last used row in a column
- **Core module: GetLastColumn** — find and copy the last used column in a row
- **Core module: UnhideRows** — unhide all rows on the active sheet in one click
- **Pro module: AdvancedData** — range profiling, type breakdown, duplicate detection
- **Pro module: BulkOperations** — bulk text transformations with regex support
- **Pro module: DataExport** — export ranges to CSV, TSV, JSON, Markdown
- **Pro module: SqlBuilder** — generate INSERT/CREATE TABLE SQL from sheet data
- Online licence validation with HMAC-SHA256 signed entitlements
- Machine activation and seat management via API
- 7-day offline grace period for interrupted connectivity
- Stripe-powered Pro Monthly and Pro Annual subscriptions
- Customer billing portal for self-service subscription management
